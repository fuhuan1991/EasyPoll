
# EasyPoll

This is a polling application. You can start a poll without any account creation.

All you need to do is:
1. Setup questions(single choice or multiple choice).
2. Share links with your friends
3. Watch the result.

## Tech Stack
- Spring boot
- Thymeleaf
- PostgreSQL
- Flyway
- BootStrap
- React
